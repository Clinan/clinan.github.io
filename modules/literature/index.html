# 诗

TIP

简介：花花的诗，用的每一个字，遣的每一个词，都是白大字，就如同这些字词雕嵌出地诗词那般的朴素。一个生活中的细节，一件小事， 这些都是千千万万的‘人’每天都在经历的事。读花花的诗，就像读自己的生平。 读着读着会陷入沉思，又会在不知觉中湿润了眼角。等到幡然醒悟的时候，回过头来， 就会发现这些诗写的是如此精辟，是如此的引人沉醉。
